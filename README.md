## Запуск проекта

### Требования
- JDK 25
- PostgreSQL (установленный локально)

### Шаг 1 — Создать базу данных

Открыть pgAdmin или psql и выполнить:

```sql
CREATE DATABASE studentforge;
```

### Шаг 2 — Настроить подключение

Открыть файл `src/main/resources/application.yaml` и при необходимости поменять пароль:

```yaml
spring:
  datasource:
    url: jdbc:postgresql://localhost:5432/studentforge
    username: postgres
    password: 12345  # ← поменять если у вас другой
```

### Шаг 3 — Запустить приложение

Открыть файл `src/main/java/com/studentforge/StudentForgeApiApplication.java` и нажать кнопку **Run** рядом с методом `main`.

В консоли появится ссылка — приложение запущено.

При первом запуске Liquibase автоматически создаст все таблицы и наполнит их тестовыми данными — ничего делать вручную не нужно.

### Шаг 4 — Открыть в браузере

| Что | Адрес |
|-----|-------|
| Веб-интерфейс | http://localhost:8080 |
| Swagger UI (API) | http://localhost:8080/swagger-ui.html |
| Health check | http://localhost:8080/actuator/health |

## Как использовать Swagger?

Swagger — встроенный инструмент для тестирования API прямо в браузере, без Postman.

1. Открыть http://localhost:8080/swagger-ui.html
2. Найти эндпоинт `POST /api/v1/auth/login`, нажать **Try it out**, вставить:
```json
{
  "email": "admin@gmail.com",
  "password": "admin123"
}
```
3. Нажать **Execute**, скопировать `accessToken` из ответа
4. Нажать кнопку **Authorize** вверху страницы (🔒), вставить токен → **Authorize**
5. Теперь все запросы авторизованы

## Сущности (9 штук)

| Сущность | Описание |
|----------|----------|
| `User` | Пользователь системы (студент или администратор) |
| `StudentProfile` | Профиль студента (университет, специальность) |
| `AcademicPeriod` | Учебный период (семестр) |
| `Group` | Учебная группа, привязанная к периоду |
| `Enrollment` | Зачисление студента в группу |
| `Subject` | Учебный предмет группы |
| `Assignment` | Задание по предмету (лаб, тест, экзамен, проект) |
| `Submission` | Сданная студентом работа |
| `Grade` | Оценка за задание |

| GET / PUT / DELETE | `/periods/{id}` | Просмотр / изменение / удаление |
| GET / POST | `/groups` | Список / создание групп |
| GET / PUT / DELETE | `/groups/{id}` | Просмотр / изменение / удаление |
| GET | `/groups/{id}/students` | Список студентов в группе |
| GET / POST | `/subjects` | Список / создание предметов |
| GET / PUT / DELETE | `/subjects/{id}` | Просмотр / изменение / удаление |
| GET / POST | `/assignments` | Список / создание заданий |
| GET / PUT / DELETE | `/assignments/{id}` | Просмотр / изменение / удаление |
| POST | `/enroll` | Зачислить студента в группу |
| DELETE | `/unenroll` | Отчислить студента из группы |
| GET | `/users/{userId}/groups` | Группы конкретного студента |
